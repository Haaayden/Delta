# How To Run
1. 安装依赖
`$ npm install`
2. 开发
`$ npm run dev`
控制台提示 Server is listening for http://localhost:3030 代表服务启动成功

public 下的文件夹代表页面路径
如访问xssmobile的页面,在浏览器中输入http://localhost:3030/public/xssmobile/index.html即可访问

修改完代码保存后会自动重新编译,但是需要手动刷新浏览器查看效果
